# Instagram UI Clone

Features:

 1) Story Feature
 2) Social Scroll
 3) Clean Code

# Setup

Clone the Repository:

git clone https://github.com/your-username/your-repository.git

Navigate to the Project:

cd your-repository

Install Dependencies:

flutter pub get

Run the App:

flutter run
